### 14.1 前台可视化编辑（所见即所得）

管理员在前台直接点击文字即可编辑，无需进入后台。

**新增文件：**
- [src/components/cms/EditableText.tsx](file:///d:/上山采月亮的台式/作品集网站/2026-作品集网站/src/components/cms/EditableText.tsx) — 可编辑文字组件
- [src/app/api/admin/check/route.ts](file:///d:/上山采月亮的台式/作品集网站/2026-作品集网站/src/app/api/admin/check/route.ts) — 管理员状态检测 API

**修改文件：**
- [src/app/api/text-content/route.ts](file:///d:/上山采月亮的台式/作品集网站/2026-作品集网站/src/app/api/text-content/route.ts) — 新增 PATCH 方法（管理员鉴权 + 更新内容）
- [src/components/cms/DynamicText.tsx](file:///d:/上山采月亮的台式/作品集网站/2026-作品集网站/src/components/cms/DynamicText.tsx) — re-export EditableText

**功能要点：**
- 管理员登录后，前台文字显示铅笔图标 + 虚线边框
- 点击铅笔进入 contentEditable 模式
- 失焦或 Ctrl+S 保存，调用 PATCH API
- 保存成功显示绿色对勾 2 秒，失败恢复原内容
- 用 `innerText` 读取内容，防止 XSS
- 非管理员完全无编辑入口

## 十五、已知注意事项

1. **Supabase 项目 ID**：`hnujowombcgfxledpnxe`，不要把 SQL 执行到别的项目
2. **管理员角色元数据**：`raw_app_meta_data` 中 `role` 必须用直引号 `"admin"`
3. **CRON_SECRET**：部署前必须在 Vercel 设置，否则定时发布无法工作
4. **dnd-kit 传感器**：移动端需长按 500ms 才触发拖拽，避免误触
5. **评论审核**：用户提交的评论默认 `is_approved=false`，需在后台审核后展示
6. **打印简历**：`/resume/print` 页面用 `window.print()` 触发，用户可选择「保存为 PDF」
7. **3D 粒子懒加载**：Three.js 通过 `next/dynamic` 懒加载，首屏不初始化 WebGL
8. **字体优化**：使用 `next/font/google` 的 Geist 字体，`display: swap` 避免文字闪烁
