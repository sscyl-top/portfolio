## 媒体上传问题修复（核心进展）

### 问题1: alt_text NOT NULL 约束违反
- **现象**: 上传图片显示"数据库记录保存失败: null value in column 'alt_text'"
- **原因**: API 路由中 `alt_text: altText || null`，用户不填替代文本时传 null
- **修复**: 改为 `alt_text: altText || file.name`，用文件名作为兜底
- **影响文件**: `src/app/api/media/upload/route.ts`, `src/app/admin/(protected)/media/actions.ts`
