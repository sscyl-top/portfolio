## 首页 Hero 浮动卡片实心化

**问题：** 用户反馈首页第一屏三个小卡片（`FloatingImageCard`，`HeroShowcase.tsx`）看着是透明的，融进黑色背景。

**根因：**
1. `bg-black` 兜底色和页面黑色背景完全同色 → 边界消失
2. `grayscale` 默认全灰 → 渐变层（mono/warm/graphite）饱和度被压到最低
3. 阴影和边框设置过弱

**改动（`src/components/home/HeroShowcase.tsx` line 290-315）：**
- 移除 `grayscale` 和 `hover:grayscale-0`
- `bg-black` → `bg-zinc-900`（比页面背景略亮，做出"实体"感）
- `border-white` → `border-white/20`（柔和不刺眼）
- `shadow-black` → `shadow-black/70`（投影更深，立体感）
- `hover:scale-[1.02]` 替换 `hover:grayscale-0`
- 加 `ring-1 ring-inset ring-white/5` 内描边让边缘有"厚度"

**同步改测试（`HeroShowcase.test.tsx`）：**
- 原断言 `grayscale` + `hover:grayscale-0` 改为 `bg-zinc-900` + `hover:scale-[1.02]` + 验证不含 `grayscale`
- 测试 2/2 通过

## 第二轮：为什么视频卡还是"透明"

**排查：** 拉线上 HTML 发现 3 张小卡片都配置了视频（sideCard1/2/3 都有 MP4 URL），不是渐变占位。

**根因：** 视频本身可能是黑底/暗场内容，铺满卡片后让卡片看起来像融进黑色背景。`bg-zinc-900` 被视频盖住，根本看不到。

**新改动（`src/components/home/HeroShowcase.tsx`）：**
- 边框 `border-white/20` → `border-white/50`（白色边框更明显，黑底视频也能框出卡片）
- 阴影改为白色外发光：`shadow-[0_0_30px_-8px_rgba(255,255,255,0.18)]`（黑色投影在黑底上看不见）
- 内描边 `ring-white/5` → `ring-white/10`

**如果还不满足"完全不透明"：** 说明视频内容是黑底。可选方案：
1. 后台 /admin/hero 取消这 3 张小卡片的视频，恢复为不透明渐变占位
2. 重新导出视频时带非黑背景
3. 改成 `object-contain` 让背景在视频四周露出来

## 第三轮：视频不再满铺 — 卡片加可见边框 + 内缩留边

**根因确认：** 3 张小卡片视频铺满卡片且视频黑底，与页面 `#050505` 背景同色，看上去透明。

**改动（`src/components/home/HeroShowcase.tsx` line 289-315）：**
- 边框：`border border-white/50` → `border-2 border-white/80`（2px 白框，不是 1px）
- 外发光：`shadow-[0_0_30px_-8px_rgba(255,255,255,0.18)]` → `shadow-[0_0_40px_-10px_rgba(255,255,255,0.2)]`
- **视频不再铺满**：`absolute inset-0 w-full h-full object-cover` → `absolute inset-1 object-cover rounded`（内缩 4px，露出 `bg-zinc-900` 框边）
- 渐变占位同样内缩：`absolute inset-0` → `absolute inset-1 rounded`
- 移除了 `h-full w-full`（`inset-1` 已同时设定定位和尺寸）

**测试：** 2/2 通过（内联 style `backgroundColor === "rgb(37, 37, 37)"` + 不含 `grayscale`）

## 第四轮：最终修复 — 内联样式彻底实心化

**根因确认（通过 agent-browser 截图验证）：**
- 左上卡片：有暖色调视频，勉强可见
- 左侧/底部卡片：**视频内容几乎纯黑**，与页面 `#050505` 同色
- 即使 inset-1 + border-2 也无法解决——4px 太窄

**最终方案（`src/components/home/HeroShowcase.tsx`）：**
- 放弃 Tailwind class（可能被其他规则覆盖或优先级不够）
- **全部用 inline style**：
  - `backgroundColor: "#252525"` — 比页面 #050505 明显更亮
  - `border: "2px solid rgba(255,255,255,0.85)"` — 白色粗边框
  - `boxShadow: "0 0 48px -12px rgba(255,255,255,0.22)"` — 白色外发光
- 视频 **内缩 10px**（不是 4px），保证四周露出实心底色
- 测试断言改为检查 `style.backgroundColor`
